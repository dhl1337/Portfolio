FAVOUR

ABOUT

Favour has a thread that people can ask for a favor or offer services to others. If there is a post that you are interested in by clicking the star it will save the post into your “favorite” tab. This way later on if you want to come back, it will always be saved since it can get lost in the thread. Each individual has their own profile, which allows others to rate and review their experiences with each other. Everyone can use a hand every once in awhile, therefore with the favour website it makes interacting with the community convenient.

RESPONSIBILITY

Before getting started on the project I listed what the website would have. Once I finished writing it out I put all my ideas into a wire frame. This made it easier to visualize exactly what I wanted to project to look like. There were new technologies that I was interested in applying on my project; therefore I began researching more about it. I began writing the code and upon finishing it I hosted my website for others to review it and give feed backs. Lastly, I fixed any bugs that were causing my page to not work.

TECHNOLOGY
